Redmine Team Board

Get a quick overview what each of your team members is working on!

Installation

  1. Clone the repository in your Redmine installation's plugin folder: git clone https://github.com/lawitschka/redmine_team_board.git
  2. Restart Redmine to pick up the plugin
  3. Go to the Plugin settings and chose which ticket statuses should be displayed on the team board
  4. Active the plugin on a per project basis in the project module settings
